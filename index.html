Get Started

Get Started

HiGHS is a high performance serial and parallel solver for large-scale sparse linear programming (LP) problems developed in C++11. HiGHS is based on the dual revised simplex method implemented in HSOL, which was originally written by Qi Huangfu. Features such as presolve, crash and advanced basis start have been added by Julian Hall, Ivet Galabova. Other features, and interfaces to C, C#, FORTRAN, Julia and Python, have been written by Michael Feldmeier.

HiGHS is freely available under the MIT licence. Installing from source code requires cmake minimum version 3.0.0. HiGHS can be used as a standalone executable on Windows and Linux. There is a C++11 library which can be easily used within a C++ project.

Documentation can be built with doxygen. Download the latest revision of HiGHS from here and call ./doxygen in HiGHS/docs . Load HiGHS/docs/index.html into any web browser for descriptions of methods. Note, that we are currently working to improve our documentation. Guide

This guide sets out what HiGHS can be used for, and how to run it, either from a command line or when called within another application.

Specification

HiGHS can load LP and MIP models from either data files, or via data provided by another applicaiton. It can solve LP problems using the dual revised simplex method and MIP problems using branch-and-bound. The solution can be written to a file or retrieved directly for use within an applicaion. When used within an application, HiGHS can be used to modify the current problem and re-solve it efficiently.
